# StackExchange
This app fetches questions from StackOverflow.

## Key Technologies
1. MVVM Architechture
2. Retrofit2 Library
3. LiveData and ViewModels
4. StackExchange REST API
5. Room persistance library
6. Google Material Design
7. RecyclerView
8. Gson Library
